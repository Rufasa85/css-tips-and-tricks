Css Tips and Tricks!

Check out these sweet css snippets! I made them my self. I am such a proud papa!

Reset

Browsers like to apply their own stylesheets, meaning your code can look different on differet browsers. This snippet will lightly reset that.

            
* {
    margin:0;
    padding:0;
    box-sizing: border-box;
}

Flex stuff

Want all your items to line up nicely? try flexbox!

                    
main{
    background:#c0ffee;
    min-height: 70vh;
    padding:1%;
    display: flex;
    flex-wrap:wrap;
    justify-content: space-around;
}
